# Instantale World Viewer

Instantale のセーブデータから「エリア接続相関図」を表示する Web アプリです。
セーブをドラッグ&ドロップするだけで、ワールドのエリアどうしのつながりや、各エリアの施設を一目で確認できます。

## 使い方

次のどちらかで開きます。

- 公開 URL（GitHub Pages）にアクセスする。
- `instantale_world_viewer.html` をダウンロードし、ダブルクリックで開く（ローカルでそのまま動きます）。

開いたら、**Instantale のセーブを画面にドラッグ&ドロップ**（またはクリックして選択）するだけです。
パスワードやパスフレーズの入力は不要で、すぐにエリア接続図が表示されます。

## できること

- エリアどうしの接続を矢印（双方向 ↔）で表示します。
- 開始地点をリングとバッジで強調し、開始からの距離で色分けします（到達できないエリアは灰色）。
- **エリアをクリックすると、その場に詳細を表示**します。
  - エリア名 / つながっているエリア / 施設・ダンジョンの一覧。
  - 施設は入口から辿れる順に**階層（ツリー）**で並びます。
- 同じエリアを続けてクリックすると、そのエリアにフォーカス（拡大）します。
- エリアどうしは重ならないよう自動配置されます。エリアはドラッグで動かして固定できます。
- 背景のドラッグで移動、マウスホイールで拡大・縮小、「フィット」「再配置」ボタンで全体表示・並べ直しができます。
- 施設の見せ方は「一覧パネル」⇄「図の上に表示（□=施設 / ◇=ダンジョン）」で切り替えられます。

## プライバシー

- セーブの読み込み・復号・解析は、すべてブラウザのメモリ上だけで行います。
- データがサーバへ送信・保存されることはありません。
